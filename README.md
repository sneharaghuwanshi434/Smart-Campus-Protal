# 🎓 Smart Campus Portal

## 📌 Project Overview

Smart Campus Portal is a web-based application designed to manage and simplify campus-related activities such as announcements, student resources, faculty information, and communication.

This project is being developed step-by-step as part of my full-stack web development learning journey.

---

## 🎯 Objective

The goal of this project is to:

* Build a real-world structured web application from scratch
* Apply frontend and backend development concepts
* Create a scalable system that can be extended into a full-stack solution

---

## 🚀 Current Features (Phase 1 - HTML)

* Multi-page website structure
* Semantic HTML implementation
* Navigation bar for easy access
* Sections for:

  * Announcements
  * Features
  * Contact information
